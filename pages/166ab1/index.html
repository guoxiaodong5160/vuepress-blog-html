item2免密ssh登录

# 查看项目结构,tree

# 1. ssh携带pom远程登录

# 方法1: 直接登录

ssh -i /Users/yuyu/.ssh/123/yuyu.pem yuyu@ops.com
1

登录后需要输入密码

# 方法2: 免密登录

// 添加pom到ssh中
ssh-add -k /Users/yuyu/.ssh/123/yuyu.pem
// 提示输入密码, 成功
// 下一步直接ssh登录即可
ssh yuyu@ops.com
